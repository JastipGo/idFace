Face Auth

Website satu file untuk daftar, simpan wajah terenkripsi di browser, lalu masuk hanya dengan deteksi wajah. Alurnya dibuat sederhana agar bisa dijalankan langsung dari satu file HTML.

🔐 Web Crypto API 📷 Webcam 🧠 Face Recognition 💾 localStorage
Catatan penting
Aplikasi browser-only bisa menyimpan data terenkripsi di perangkat, namun keamanan penuh tetap bergantung pada lingkungan browser dan perangkat yang dipakai.
Masuk memakai wajah. Sistem akan membandingkan descriptor wajah kamera dengan data wajah yang tersimpan.
Status: menunggu model dan kamera siap.
Saat daftar, isi nomor HP, password, lalu ambil sampel wajah beberapa kali supaya hasil pengenalan lebih stabil.
Status: belum ada sampel wajah.
